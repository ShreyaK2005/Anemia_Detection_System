# Anemia_Detection_System

🔹 Project Title: AI-Based Anemia Detection System

🔹 Problem: Anemia is often undiagnosed non invasively. Blood tests are invasive
🔹 Solution: Detect anemia using image processing (palpebral conjunctiva images of eye + symptoms)
🔹 Features
    Non-invasive detection
    Image + symptom fusion
    Grad-CAM visualization
🔹 Tech Stack: Python, PyTorch, OpenCV, Tkinter
